# TheFinalCountdown

I was frustrated with how online timers, countdowns and stopwatches did not have enough features. For personal time tracking and productivity, I needed something that could count up, down, stop at any time, and have a dynamically controlled speed multiplier. I also wanted it to sync across my devices.

So I decided to do it! I made a simple counter with those features using HTML, CSS and JavaScript that connects to Firebase to maintain anonymous sessions and synchronize the counting across devices with very high accuracy. It's live at <https://luizfneves404.github.io/TheFinalCountdown/>!
